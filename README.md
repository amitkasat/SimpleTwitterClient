Overview: This is a simple Twitter client app that supports viewing a Twitter timeline and composing a new tweet.

Total Hours spent:- 25 hours Willing to spend another:- 5-7 hours to finish some of the optional parts like pull to refresh and limiting characters.

User Stories:-

User can sign in to Twitter using OAuth login.

User can view the tweets from their home timeline.

User should be able to see the username, name, body and timestamp for each tweet.

User should be displayed the relative timestamp for a tweet "8m", "7h" User can view more tweets as they scroll with infinite pagination.

User can compose a new tweet User can click a “Compose” icon in the Action Bar on the top right.

User can then enter a new tweet and post this to twitter.

User is taken back to home timeline with new tweet visible in timeline.

Optional: User can see a hint with total number of characters left for tweet.

Optional: Pull to Refresh is working now
